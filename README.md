# Natural Language Processing (DSA4020)

This repository contains classwork, laboratory exercises, assignments, and projects completed during my **DSA4020: Natural Language Processing** course in **Summer 2026 (May–August)**.

The repository serves as a record of my learning journey in Natural Language Processing (NLP), covering both foundational concepts and modern deep learning approaches for language understanding and generation. 


## Topics Covered

* Introduction to Natural Language Processing
* Text Preprocessing
* Language Modeling
* Part-of-Speech (POS) Tagging
* Syntax and Parsing
* Semantics
* Machine Translation
* Information Retrieval and Text Mining
* Sequence Labeling and Named Entity Recognition (NER)
* Topic Modeling
* Advanced Neural Networks for NLP
* Ethics and Bias in NLP



## Tools and Libraries

Throughout this course, I will be working with:

* Python

* NLTK
* spaCy
* Scikit-learn
* PyTorch
